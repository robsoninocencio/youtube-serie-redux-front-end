## Projeto -> Desvendando o Redux
Essa série tem como objetivo refatorar uma aplicação de Shopping Car adicionando o Redux passo a passo. Utilizando ReactJS como biblioteca principal, React-Router-Dom para tratamento de rotas, Redux responsável pelo meu estado e React-Redux para Ligações oficiais.
